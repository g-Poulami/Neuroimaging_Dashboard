# Paediatric Brain Tumour Neuroimaging Research Dashboard

An interactive data visualisation dashboard built to explore simulated MRI/MRS 
research data inspired by my MSc Bioinformatics dissertation at the University 
of Birmingham in collaboration with Birmingham Children's Hospital.

## What it shows
- Tumour grade distribution across 120 paediatric patients
- MRS metabolite profiles (Cho, NAA, Cr) by tumour grade
- Cho:NAA ratio vs survival scatter plot (Favourable vs Poor outcome)
- Median survival analysis by grade
- Tumour volume comparisons and age distribution
- Live filters by grade, tumour type, and outcome

## Built with
- HTML, CSS, JavaScript
- Chart.js for interactive visualisations
- Python (NumPy, Pandas) for dataset generation
- Tableau Public for the BI dashboard version

## Background
This project was built independently to develop hands-on BI dashboard skills, 
extending computational work from my MSc research where I developed Python and 
R pipelines for clinical neuroimaging analysis.
